# ConvNeXt-Base Image Classifier — Streamlit Dashboard
This dashboard provides a user-friendly interface to visualize the performance of a ConvNeXt-Base image classifier trained on 12 rice varieties. It displays key metrics such as precision, recall, F1-score, and support for each class, along with overall macro averages.
## Usage
1. Ensure you have Streamlit installed: `pip install streamlit`
2. Make sure the `best_model_finetuning.pt` file is in the same directory.
3. Run the dashboard from terminal: `streamlit run convnext_dashboard.py`
